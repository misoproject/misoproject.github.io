If you've written a d3.chart based chart that you would like to have listed on the Miso Project, please submit it using this form. This is a temporary solution while we resolve a better distribution system.


There are two requirements:

  • 1. Please name your chart in the format: d3.chart.*, for example d3.chart.barchart etc.
  • 2. Please add a section to your package.json that looks like this: { "d3.chart" : { "version" : "0.1.2" } }

For more guidelines, see the following wiki page: https://github.com/misoproject/d3.chart/wiki/chart-contribution
