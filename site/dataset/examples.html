Cabinet Office Spending

This example uses d3 and backbone.js to build an interactive treemap that allows you to explore UK Government Cabinet Office spending in the 2010/2011 financial year

This example demonstrates: grouping & selecting data, remote data files

Remote CSV Dataset with Highcharts

This example pulls a local comma delimited dataset containing world crude oil production since 1980 to 2004 and graphs it as a line chart using Highcharts.

This example demonstrates: CSV Delimited Data, Time formatting

Remote Polling Twitter & Rendering With Rickshaw

Remotely polls twitter and plots a stacked bar chart (using Rickshaw) that shows the use of punctuation in tweets for a particular query.

This example demonstrates: Polling, Group By, Binding.

jQuery Sparklines with Github API

Remotely fetches commit history for the Dataset project, builds a timeseries with a week long interval and renders a basic line chart.

This example demonstrates: Remote Importing, countBy and column 'before' coersion callbacks.

Perlin Noise Generation and DOM based rendering

Creates a matrix of perlin noise values and renders them progressively using DOM elements only.

This example demonstrates: Creating a Dataset from scratch.
